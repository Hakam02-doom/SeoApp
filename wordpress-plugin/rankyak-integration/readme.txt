Connect your WordPress site with RankYak to automatically publish SEO-optimized articles.

== Description ==

RankYak Integration allows you to automatically publish SEO-optimized articles from RankYak to your WordPress site. The plugin provides:

* Integration key authentication for secure connection
* Automatic article publishing
* Real-time webhook support
* Two-way synchronization

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rankyak-integration` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Get your integration key from RankYak dashboard (Settings → Integrations)
4. Go to RankYak > Settings in your WordPress admin
5. Enter your integration key and click "Connect"
6. Start publishing articles from RankYak!

== Frequently Asked Questions ==

= How do I connect my WordPress site to RankYak? =

1. Get your integration key from RankYak dashboard (Settings → Integrations)
2. Go to RankYak > Settings in your WordPress admin
3. Enter your integration key and click "Connect"

= Can I use this with WordPress.com? =

This plugin is designed for self-hosted WordPress installations. For WordPress.com, use the REST API integration method.

= How do I disconnect? =

Go to RankYak > Settings and click the "Disconnect" button.
